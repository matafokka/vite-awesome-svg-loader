<Aside>
  This imports raw base64 string. If you're looking for [data URI](https://en.wikipedia.org/wiki/Data_URI_scheme), use
  `?base64-data-uri` import.
</Aside>

SVGs may be imported as base64 strings. This is done by appending `?base64` to the import:

```ts
import imageUrl from "./image.svg?base64";
```

A line width of an image may be [preserved](/guides/available-transformations/#preserve-line-width)
by adding a `preserve-line-width` parameter to the import query:

```ts
import imageUrl from "./image.svg?base64&preserve-line-width";
```

Other transformations may be applied via [configuration](/loader-api-reference/interfaces/svgloaderoptions).
