<Aside type="caution" title="Consider alternative approaches">
It's easy to make mistakes while working with elements and especially easy when using allow- and ignore-lists.

Moreover, some options introduce heavy build time impact.
</Aside>

It's possible to skip transforms on per-element or per-element-per-file basis via following
[configuration options](/loader-api-reference/interfaces/svgloaderoptions):

1. [skipPreserveLineWidthSelectors](/loader-api-reference/interfaces/svgloaderoptions/#skiptransformsselectors) -
a list of selectors to skip while preserving line width.

1. <Badge variant="caution" text="Build time impact" /> [skipReplaceColorsSelectors](/loader-api-reference/interfaces/svgloaderoptions/#skipreplacecolorsselectors) -
a list of selectors to skip while replacing colors.

1. <Badge variant="caution" text="Build time impact" /> [skipTransformsSelectors](/loader-api-reference/interfaces/svgloaderoptions/#skiptransformsselectors)
- a list of selectors to skip while transforming.

The demo below utilizes all these options. See `vite.config.ts` and `src/assets/ignore-elements.svg` files.
