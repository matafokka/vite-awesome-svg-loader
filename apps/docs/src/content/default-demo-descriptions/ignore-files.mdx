There are following [configuration options](/loader-api-reference/interfaces/svgloaderoptions) to set up ignore lists on
per-file basis:

1. [skipPreserveLineWidthList](/loader-api-reference/interfaces/svgloaderoptions/#skippreservelinewidthlist) -
files where line width should **NOT** be preserved

1. [skipReplaceColorsList](/loader-api-reference/interfaces/svgloaderoptions/#skipreplacecolorslist) -
files where colors should **NOT** be replaced.

1. [skipTransformsList](/loader-api-reference/interfaces/svgloaderoptions/#skiptransformslist) -
files to **NOT** transform (except SVGO).

1. [skipFilesList](/loader-api-reference/interfaces/svgloaderoptions/#skipfileslist) -
files to bypass. They will be passed to another loader.
