`SvgImage` uses SVG symbols to improve performance by reducing overall amount of DOM nodes. `SvgImage` offers
customization that covers most common use cases. See [SVG loading methods](/guides/svg-loading-methods) for more
details.

This example introduces `SvgImage` and additional `vite-awesome-svg-loader`
[configuration](/loader-api-reference/interfaces/svgloaderoptions). See `vite.config.ts` file in the demo's files below.
