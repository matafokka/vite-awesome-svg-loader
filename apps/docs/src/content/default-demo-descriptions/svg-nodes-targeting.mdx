Sometimes, it is required to change a specific SVG node's style, or make it interactive. In other words, nodes targeting
is required.

This is possible with `vite-awesome-svg-loader` with one condition: all IDs and classes of every SVG elements
are prefixed with a unique string to avoid collisions.

This prefix can be extracted like so:

```ts
import imageSrc, { prefix: imagePrefix } from "@/image.svg";
```

Then a node can be selected like so:

```ts
document.getElementById(imagePrefix + "some-svg-node");
```

The targeting is implemented by accessing nodes with their class names.
