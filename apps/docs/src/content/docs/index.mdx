This is a framework-agnostic Vite plugin that:

1. Can [import SVGs](/guides/svg-loading-methods/) as:
   1. Source code (default import type).
   1. URL to a static asset.
   1. Source code [data URI](https://en.wikipedia.org/wiki/Data_URI_scheme).
   1. Source code base64.
   1. Source code base64 [data URI](https://en.wikipedia.org/wiki/Data_URI_scheme).
1. Can [improve performance](/guides/svg-loading-methods/) of your website by using an alternative SVG loading method.
1. Can [preserve line width](/guides/svg-transformations/#preserve-line-width). Images will retain line width
regardless of the size.
1. Can replace colors with [currentColor](/guides/svg-transformations/#replace-all-colors-with-currentcolor-value)
or a [custom color via configuration](/guides/svg-transformations/#replace-colors-with-custom-values).
1. Automatically minimizes your SVGs with [SVGO](https://github.com/svg/svgo).
1. Allows you to create [SVG sprites](/guides/svg-loading-methods/#svg-symbols-sprites) using optional integrations.
