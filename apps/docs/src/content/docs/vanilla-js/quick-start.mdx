<Aside type="caution" title="Not for frameworks integrations">
  Do **NOT** use this to integrate with a framework. The result almost always will be bulkier than a specialized
  integration. Use `vite-awesome-svg-loader/integration-utils` package instead.
  See [other frameworks guide](/other-frameworks/quick-start/).
  ___

  Only use Vanilla JS integration if you need its complete API for your own integration.
</Aside>

<Steps>
1. Import classes:

   ```ts
   import {
     SvgImage, // Renders SVG sprites
     SvgIcon, // Basic SVG icon that uses SvgImage class internally
   } from "vite-awesome-svg-loader/vanilla-integration";
   ```

1. Import images:

   ```ts
   import imageSrc from "@/path/to/image.svg";
   ```

1. Display images:

   ```ts
   new SvgImage(imageSrc, "#my-container"); // Create an image and mount it to the element with "my-container" id.

   new SvgIcon(imageSrc, "#my-container") // Create an icon and mount it to the element with "my-container" id.
     .setSize("24px") // Set 24px size
     .setColor("red") // Set red color
     .setColorTransition("0.2s ease-out"); // Set color transition
   ```
</Steps>
