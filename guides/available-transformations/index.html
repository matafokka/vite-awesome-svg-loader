Available SVG transformations

Preserve line width

Section titled “Preserve line width”

This makes all SVG elements with stroke have constant line width regardless of the image size.

This is done via ?preserve-line-width import or preserveLineWidth option.

Replace all colors with currentColor value

Section titled “Replace all colors with currentColor value”

This allows you to recolor your images with a single color.

This is done via ?set-current-color import or replaceColorsList option.
