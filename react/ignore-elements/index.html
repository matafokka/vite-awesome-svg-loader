Ignore elements

Tip

You probably don’t need these options because it’s easy to make mistakes while working with elements and especially easy when there’re allow and ignore lists.

Think of the other ways to solve your problem. See the demos for the tips.

It’s possible to skip transforms on per-element or per-element-per-file basis via following configuration options:

  1. skipPreserveLineWidthSelectors - a list of selectors to skip while preserving line width.

  2. Build time impact skipReplaceColorsSelectors - a list of selectors to skip while replacing colors.

  3. Build time impact skipTransformsSelectors

  • a list of selectors to skip while transforming.

The demo below utilizes all these options. Make sure to check out vite.config.ts and src/assets/ignore-elements.svg files.
